JavaScript Image Slider

A responsive and interactive image slider built with HTML, CSS, and Vanilla JavaScript. This project allows users to navigate through images using previous/next controls and navigation dots with a smooth fade animation.

Features
Responsive image slider
Previous and Next navigation buttons
Navigation dots for direct slide selection
Smooth fade transition animation
Dynamic slide generation using JavaScript
Clean and simple user interface
Technologies Used
HTML5
CSS3
JavaScript (ES6)
